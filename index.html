Ответы на вопросы

  1. Сколько детей (дочерних DOM-узлов) у элемента ul в примере ниже:
    ul
    li Привет li
    li Мир li
    ul

    2. Ответ: 2;


  2. Есть такой элемент: input id= "input" value = "Привет". Какой вызов поменяет значение в нём?

    3. Ответ: let value = input.setAttribute('value', 'Пока'); ==> input id= "input" value = "Пока";


  3. Сколько потомков будет у div после кода ниже?
    div id="div" /div
    p id="p" Привет /p
    script
    div.append(p);
    div.append(p);
    script

    4. Ответ: дочерний элемент p и текстовый узел;


  4. В каких случаях для элемента elem не верно elem.firstChild == elem.lastChild?
    - Когда у elem два или более потомков
    - Когда у elem единственный потомок
    - Когда у elem нет потомков
    - Нет такого варианта ответа

    5. Ответ: Когда у elem нет потомков;Когда у elem единственный потомок;


  5. Какое свойство задает класс элемента?

    6. Ответ: classList.add()


  6. Для чего используется запись: setAttribute(name, value)?

    7. Ответ: Устанавливает значение атрибута;


  7. Назовите метод добавления текста к документу.

    8. Ответ: Свойство textContent позволяет считывать или задавать текстовое содержимое элемента;


  8. Как получить HTML-содержимое DOM-элемента elem?

    9. Ответ: Свойство innerHTML содержит в себе строку со всем наполнением элемента (в том числе и разметкой);


  9. Что такое объект события и какие у него могут быть свойства?

    10. Ответ: Получить детальную информацию о событии в обработчике можно посредством объекта события - Event;


  10. Что такое BOM и DOM?

    11. Ответ: Browser Object Model (BOM) - объектная модель браузера. Основные объекты Browser Object Model: window, navigator, history, location, screen, document. Document Object Model (DOM) — это объектная модель документа, которую браузер создает в памяти компьютера на основании HTML-кода;


  11. Как найти:
    Таблицу с `id="age-table"`
    Все элементы `label` **внутри** этой таблицы (их три)
    Форму `form` с именем `name="search-person"`

    Ответ:
    const table = document.getElementById('age-table');
    const label = document.querySelector('#age-list');
    const form = document.getElementsByName('search-person');

  12. Как сделать переход на другую страницу при клике на кнопку (без a href=...>, только средствами JavaScript)?

    13. Ответ:
    function click() {
    location.href = "https:///";
    }
